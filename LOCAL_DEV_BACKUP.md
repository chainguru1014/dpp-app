# Local Dev Backup (App)

## Setup

1. Copy `env.sample` to your local env file.
2. Install dependencies: `npm install`

## Run

- Mobile Metro: `npm start`
- Web dev server: `npm run web` (port `3001`)
- Static web server: `npm run serve:web` (default port `3002` or host `PORT`)

Use backend API at `http://localhost:5052` for local development.
