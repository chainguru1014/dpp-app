# Mobile App Setup

## Installation

1. Install dependencies:
   ```bash
   cd app
   npm install
   ```

2. For iOS (if applicable):
   ```bash
   cd ios
   pod install
   cd ..
   ```

## Configuration

Update the API base URL in the following files:
- `src/screens/LoginScreen.tsx`
- `src/screens/RegisterScreen.tsx`
- `src/screens/ScannerScreen.tsx`
- `src/screens/ResultScreen.tsx`

Change `API_BASE_URL` from `'http://localhost:8080'` to your actual backend URL.

For Android emulator, use: `http://10.0.2.2:8080`
For physical device, use your computer's IP address: `http://192.168.x.x:8080`

## Running the App

### Android
```bash
npm run android
```

### iOS
```bash
npm run ios
```

## Features

- **Login/Register**: Responsive design matching admin panel
- **Home Screen**: 
  - Yometel header
  - QR code scanner icon (left)
  - User avatar with menu (right)
  - Product display area with grid overlay
- **QR Scanner**: Scans QR codes generated by admin panel
- **Product Details**: Shows product information matching admin panel preview
- **Profile Management**: Edit profile and logout functionality

## Required Permissions

The app requires camera permission for QR code scanning. These are automatically requested when needed.

## Troubleshooting

### Camera not working
- Check app permissions in device settings
- Ensure camera is not being used by another app

### Network errors
- Verify backend server is running
- Check API_BASE_URL is correct
- For physical device, ensure device and computer are on same network

### Build errors
- Clear cache: `npm start -- --reset-cache`
- Clean build: `cd android && ./gradlew clean && cd ..`
